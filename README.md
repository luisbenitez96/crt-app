# CRT-APP – POC

Project for a catalog of automotive diagnostic equipment and tools

## Stack

- Next.js (App Router, TypeScript)
- TailwindCSS
- Deploy en Vercel

## Rutas actuales (stub)

- `/` – basic homepage
- `/products` – Product listing (placeholder)
- `/products/[slug]` – Product details (placeholder)
- `/about` – About the company
- `/contact` – Contact us

## Scripts

- `npm run dev` – development enviroment
- `npm run build` – Production Build
- `npm run start` – Production Server
